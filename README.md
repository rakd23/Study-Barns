# StudyBarns

Interactive Chrome extension UI mockup for UC Davis students — weekly class schedule plus academic support booking.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Lucide icons

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Features

- Dual weekly calendars (Canvas classes + AATC / office hours / study sessions)
- 4-step booking modal with slot picker and Google Calendar export
- Create Study Session flow with toast share link
- Session review card with star ratings
- All state in React (resets on refresh)
